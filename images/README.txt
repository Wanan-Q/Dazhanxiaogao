========================================
  接物小游戏 - 图片资源说明
========================================

请将以下图片放入此文件夹，保持文件名一致。
图片建议为 1:1 比例（正方形），游戏会自动按比例缩放。

【必需图片】
- basket.png   : 聚宝盆（底部可移动的接物器）
- hyr.png      : 普普（+1普普）
- star_yes.png : 西达肯定（+3普普）
- gjh.png      : 小高（-10普普）
- star_no.png  : 西达否定（-1生命）

【可选图片】
- background.png : 游戏背景图（不替换则使用黑色背景）
  建议尺寸比例 9:16（竖屏，如 450×800 或 405×720），游戏使用 cover 模式铺满

【格式】
PNG 格式（代码中路径为 .png），建议 1:1 比例（正方形），游戏会自动缩放。

【说明】
当前未放置图片时，游戏会自动用彩色方块+文字占位，不影响运行。
替换图片时只需添加或覆盖对应文件，保持文件名不变即可。
